Scrollface

Scrollface is website code that you can use to show stuff in a slideshow.

Download   GitHub

Javascript

$('#slideshow').scrollface({
  next   : $('#next'),
  prev   : $('#prev'),
  pager  : $('#pager'),
  speed  : 400,
  easing : 'easeOutExpo'
});

HTML

<div id="slideshow_mask">
  <div id="slideshow">
    <div class="slide">...</div>
    <div class="slide">...</div>
    <div class="slide">...</div>
  </div>
</div>
<div id="pager"></div>
<a id="prev"></a>
<a id="next"></a>

CSS

#slideshow_mask {
  position:relative;
  overflow:hidden;
  margin:0 auto;

  position:absolute;
